# Kartchain

A web-native, Solana-powered 3D kart racer inspired by
[SuperTuxKart](https://github.com/supertuxkart/stk-code). Plays in any modern
browser, supports 8-player multiplayer, NFT karts, pay-to-race prize pools, and
play-to-earn SPL token rewards.

> **Status:** scaffold / MVP. Playable single-track multiplayer with stubbed
> Solana integration on devnet.

## Why not just fork stk-code?

SuperTuxKart is ~500k lines of C++ on a forked Irrlicht engine using ENet (raw
UDP) for networking. None of that works in a browser without months of WASM
porting, and the codebase is GPLv3 (which would force every commercial/onchain
component to also be GPL).

Kartchain is a **clean-room rewrite in Godot 4** (which has first-class WebGL2
export) that **reuses STK's free CC-BY-SA assets** (karts, tracks, music) with
attribution. Our code is MIT; reused/derived art remains CC-BY-SA.

See [`docs/LICENSING.md`](docs/LICENSING.md) for full details.

## Architecture

```
┌─────────────────────────────────────────────────────────────┐
│  Browser                                                    │
│  ┌──────────────────┐    ┌────────────────────────────┐   │
│  │ Next.js wallet   │◄──►│ Godot game canvas (WASM)   │   │
│  │ UI (React)       │ JS │ - 3D rendering             │   │
│  │ - Wallet connect │    │ - Physics, input           │   │
│  │ - NFT garage     │    │ - WebSocket to server      │   │
│  │ - Lobby          │    └────────────────────────────┘   │
│  └────────┬─────────┘                  ▲                   │
└───────────┼──────────────────────────────┼──────────────────┘
            │ HTTPS                       │ WSS
            ▼                             ▼
   ┌──────────────────┐         ┌──────────────────────┐
   │  Solana RPC      │         │ Colyseus server      │
   │  - wallet sign   │         │ - LobbyRoom          │
   │  - plain SOL tx  │         │ - RaceRoom (8p)      │
   │  - SPL transfers │◄────────┤ - tx verification    │
   └──────────────────┘  RPC    │ - holds custodial    │
            ▲                   │   wallets (escrow,   │
            │                   │   treasury, p2eMint, │
            │                   │   nftAuth)           │
            │                   │ - signs payouts +    │
            │                   │   P2E mints directly │
            │                   └──────────────────────┘
            │
   No on-chain program. Server is custodial — it holds the hot keys
   for escrow, treasury, P2E mint authority, and (optionally) NFT
   mint authority. See docs/SOLANA.md.
```

## Repo layout

| Path | Purpose |
|---|---|
| `client/` | Godot 4 project. `godot --export-release Web` ⇒ `web/public/game/` |
| `server/` | Colyseus + TypeScript authoritative server + custodial Solana wallets |
| `web/` | Next.js 15 wallet host that embeds the Godot canvas |
| `assets-import/` | Scripts to fetch + convert STK assets to glTF |
| `docs/` | Architecture, networking, Solana, licensing docs |

## Quickstart

Prereqs: Node 20, pnpm 9, [Godot 4.3+](https://godotengine.org/download).
No Rust / Anchor / Solana CLI required — keypairs are generated by a
plain Node script.

```bash
# 1. Install deps
pnpm install

# 2. Generate the four custodial keypairs (treasury, escrow, p2eMint, nftAuth)
pnpm run keys:gen

# 3. Copy env, paste your devnet RPC URL
cp .env.example .env
cp .env.example server/.env
cp .env.example web/.env.local

# 4. Start Colyseus server (port 2567)
pnpm dev:server

# 5. Build Godot client to web/public/game/
pnpm build:client

# 6. Start Next.js host (port 3000)
pnpm dev:web
```

Open http://localhost:3000 — connect a Phantom/Solflare wallet on Devnet, then
race.

See [`docs/QUICKSTART.md`](docs/QUICKSTART.md) for detailed setup including
WSL/Windows notes.

## Solana features

| Feature | Status | Notes |
|---|---|---|
| Wallet login (wallet-standard) | ✅ scaffolded | Phantom, Solflare, Backpack |
| Pay-to-race (custodial escrow) | ✅ scaffolded | Plain `SystemProgram.transfer` + memo → escrow keypair |
| P2E SPL token rewards | ✅ scaffolded | Server is mint authority; `mintTo` per finisher |
| NFT karts (Metaplex Core) | 🟡 stub | Wire `@metaplex-foundation/mpl-core` in `server/src/solana/kartNft.ts` |
| Spectator wagering | ❌ not in MVP | Future work |
| On-chain race replays | ❌ not in MVP | Future work |

See [`docs/SOLANA.md`](docs/SOLANA.md) for the full custodial design,
trust model, and migration path to an Anchor program when needed.
